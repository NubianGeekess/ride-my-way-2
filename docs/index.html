    • Overview -- Ride My Way API

    Welcome to the Ride My Way API documentation. This is the API used by the Ride My Way web interface.

    This API powers the Ride My Way app. A carpooling application that allows users create ride offers and join ride offers created by other users.

    BASE API URL: https://ride-my-way-app.herokuapp.com/api/v1/

    Making API requests

    An authorization token is required to access all endpoints except the signup and Login endpoints. Use the login/signup endpoint to get a token

    A token generated expires 24 hours after it has being generated.

    Use the token in the x-access-token field in req.headers

    x-access-token: averylongrandom.jsonwebtokenstring.requiredforauthentication

    You must replace averylongrandom.jsonwebtokenstring.requiredforauthentication with your personal token generated after authenticating.

    Respond to a Ride Offer Request

    Respond to a ride offer request

    Request Body:

    {
  "status": "accepted",
}

    Response Body:

    {
  "status": "success",
  "data": {
      "id": "ca98ccdd-d9fa-4d97-ba40-3dbb30d707c5",
      "ride_id": "75b839ea-1207-46dd-a608-4105734f574a",
      "user_id": "322b6710-7ed1-11e8-9616-c9c7fe1f2563",
      "name": "Fasoro Alex"
      "status": "accepted"
  },
  "message": "Successfully responded to ride offer request."
}

    HTTP Request

    • Endpoint: users/rides/:rideId/requests/:requestId
    • Verb: PUT
    • Possible values: accepted or rejected

    HTTP Response

    • Status code: 200: OK
    • Body: (application/json)

    Delete Ride Offer

    HTTP Response

    • Body: (application/json)

    Errors

    The Ride My Way API uses the following error codes:

    Error Code Meaning
    400 Bad Request.
    401 Unauthorized.
    403 Forbidden.
    404 Not Found.
    409 Conflict.
    500 Internal Server Error.
