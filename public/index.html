Welcome to Ride My Way API!!!

Whip out your trusty old postman to test out the following endpoints :)

Endpoint Method Request Body Purpose
api/v1/rides GET none Gets all ride offers
api/v1/rides POST driver: string, destination: string, departureTime: string, pointOfDeparture: string Creates a new ride offer
api/v1/rides/:rideId GET none Gets a single ride offer
api/v1/rides/:rideId/requests POST name: string Join a ride offer
api/v1/rides/:rideId/requests GET none Gets all requests for a ride offer
api/v1/rides/:rideId/requests POST accept: true | false Responds to a requests to join a ride offer
